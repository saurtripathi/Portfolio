Vanlifeweb app

React router
and Firebase DB

GitHub Repo Live version

This is the main layout of Vanlife - a van rental app, built using React router. It is a single page application having multiple routes where user can filter the required van based on the type of van.The app shows the rental plan for the vans listed there.Apart from this, the authenticated users can see the vans listed, the income generated, and the reviews gathered by those vans. The app applies the various react concepts such as protected routes, nested routes. Also, it is backed by the real time Firebase database.
