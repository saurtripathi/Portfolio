TenziesGaming web app

React concepts,
java script, hooks

GitHub Repo Live version

This is the main layout of Tenzies game app. The idea behind the app is gamer tries to make all dice same, for that gamer hold the die/dice which are same and rolls the dice, in subsequent rounds gamer again hold the dice which are same as previously held and roll the dice till all are same, once all are same, game is over and confetti is showered over the winner. The app records the time taken to complete the game and the best attempt which includes the number of rolls and the time taken to win the game. The app uses various React hooks, complex states, javascript concepts, third party React libraries etc.
