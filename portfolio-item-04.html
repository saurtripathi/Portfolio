Meme Generatorweb app

Interactive React web app

GitHub Repo Live version

This is the main page of Meme Generator web app. It uses various react concepts such as useState and useEffect hook .As soon as the app loads it calls the imgflip api which fetches the 100 images popular at the time. Clicking on Get a new meme image button will fetch random image, and using the input boxes we can give top and bottom text to the Meme.
